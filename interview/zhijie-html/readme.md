1. transform 是位于 Composite Layers 层
- GPU进程开启新的复合图层，不会影响默认的复合图层（就是普通的文档流）不会影响周边的DOM结构。
- 所以transform不会引起回流

3. 回流（重排）和重绘
DOM树:表示页面结构，
渲染树（DOM+CSSOM):表示DOM节点如何显示


GPU拿到渲染树后，进行布局绘制叫--重排
当页面上的DOM节点发生非布局变更时GPU需要再次绘制叫--重绘

怎样触发重排：
 - 页面首次渲染
 - 添加/删除可见的dom元素
 - 改变元素位置
 - 改变元素尺寸
 - 改变窗口大小
 - offsetWidth offset...

怎样触发重绘：
 - 当页面上的dom改变了非几何信息的属性时

回流一定重绘，重绘不一定回流
